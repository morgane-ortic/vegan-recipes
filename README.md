# VEGAN AND EASY

A website with vegan **recipes** and **articles** about **veganism**.
All coded and designed with **HTML** and **CSS**.

- Recipes from many places
- Meal plans
- Nutrition info (we called it "Lifestyle" to attract hipsters)
- No complicated recipes or ingredients
- **NO** fat-shaming. This is for helping everyone get the calories they need in forms they enjoy!

## Enjoy!
